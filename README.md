# BumbleVue

A fork of PrimeVue, born after PrimeVue was taken closed-source.
BumbleVue is a community-driven, free and open-source UI component library for Vue.js, providing a wide range of customizable and responsive components to help developers build modern web applications with ease.

## Documentation

Until a new documentation site is created, you may refer to the PrimeVue v4 documentation here: [https://v4.primevue.org/introduction/](https://v4.primevue.org/introduction/).
Note that the search functionality may redirect you to the new, closed-source version of PrimeVue. If you use the links in the sidebar, you should be OK.
